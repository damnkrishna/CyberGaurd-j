# Tools of Trade

## Penetration Testing Tools

### Metasploit
A penetration testing framework that helps security professionals find and exploit vulnerabilities in systems. It provides a collection of exploits, payloads, and auxiliary modules for testing network security.

### Burp Suite
A web application security testing tool used to identify vulnerabilities in web apps. It acts as a proxy between your browser and target application, allowing you to intercept and modify HTTP/HTTPS traffic.

### John the Ripper
A password cracking tool that uses dictionary attacks, brute force, and rainbow tables to crack password hashes. Commonly used for testing password strength and recovering lost passwords.

## Digital Forensic Tools

### tcpdump
A command-line packet analyzer that captures and displays network packets transmitted over a network. Used for network troubleshooting and security analysis.

### Wireshark
A graphical network protocol analyzer that captures and displays packet data in real-time. It allows deep inspection of network traffic and supports hundreds of protocols.

### Autopsy
A digital forensics platform for analyzing hard drives and smartphones. It helps investigators recover deleted files, analyze file systems, and extract artifacts from digital devices.

## Package Management Tools

### apt (Advanced Package Tool)
Used in Debian-based distributions like Ubuntu and Kali Linux for installing, updating, and removing software packages.

### yum (Yellowdog Updater Modified)
Used in Red Hat-based distributions like CentOS and Fedora for package management.

## Bash Commands

**command** = tell the computer what to do

### FHS (File System Hierarchy Standard)
A standard that organizes data on a computer's file system.

### File Searching Commands

- **grep** - used for searching within files
- **find** - used for searching files

### Create and Modify File Commands

- `mkdir` - create directory
- `rmdir` - remove directory
- `touch` - create empty file
- `rm` - remove file
- `mv` - move/rename file
- `cp` - copy file

## File Permission and Ownership

### Permission Types

- **read** = r
- **write** = w
- **execute** = x

### Permission Categories

- **user** - owner of the file
- **group** - group members
- **others** - everyone else

### Change Permission of Files

**chmod** = change mode of file
